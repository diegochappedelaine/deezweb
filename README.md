
# Deezweb Frontend

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Getting started

```bash
$> yarn install
$> yarn start
```

## Stack

- React
- TypeScript
- react-router-dom
- react-scroll
- react-palette
- react-h5-audio-player
- styled-components
- sass
